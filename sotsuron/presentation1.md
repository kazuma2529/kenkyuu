## 1. 研究の背景と目的

### 背景：フラン鋳型評価の課題

- **フラン鋳型の特性**: フラン樹脂を結合剤として砂と混ぜて作られる自硬性鋳型であり、内部の砂粒子の粒度分布や接触状態が、強度や透気性に影響を与えると考えられています 。

- **評価手法の限界**: 砂粒子は非球形であるため理論値の算出が難しく、従来の2D断面観察では3D的な粒子配置や接触構造を直接評価することが困難でした 。

### 目的

- X線CT画像を活用し、フラン鋳型内部の粒子サイズや接触構造を**3Dでかつ定量的に評価する手法**を確立すること 。

- 画像から砂粒子の「接触数」「体積分布」などを自動算出するシステムを開発すること 。

## 2. 提案システムと内部処理の流れ

・（作成システムのUIの画像）を貼ってこのシステムの使い方を説明

・（粒度分布を添付）して、平均直径362㎛の粒子が61.9％を占めているということを書く

開発されたシステムは、CT画像を入力変数として、砂の平均接触数や体積分布を自動出力するGUIを備えています 。主な処理プロセス（STEP1〜4）は以下の通りです。

| STEP          | 処理内容             | 詳細 |
| ------------- | -------------------- | ---- |
| **1. 二値化** | 砂粒とそれ以外を分離 |

| 大津の二値化法を採用し、白と黒の2色に塗り分ける 。

|
| **2. 粒子分割** | 接触粒子の分離

| 距離変換とWatershed法を用い、くっついて見える砂を1つずつバラバラにする 。

|
| **3. 計測** | 接触数・ラベリング

| 砂粒ごとに番号を付け、隣接する粒子との接触数をカウントする 。

|
| **4. 最適化** | 収縮回数 r の決定

| 収縮回数 を試行し、粒子数や接触数が現実的かを確認して最適な r を選ぶ 。3つのチェックポイント
１．『大きすぎる塊がないか』
２．『収縮しすぎてないか』
３．『接触数は現実的か』

|

### 結果・考察

・r = 1,2では最大粒子割合が極めて高く、分割できていない塊状態
・r=4以降は最大粒子割合が1%未満で安定し、過分割を避けられている
・r=6以降は粒子数が減少傾向でこれは過度な収縮により粒子が消失し始めたと示唆される

→内部構造としては、局所的な粗密が小さい、比較的均質な充填状態である

V = (4/3)πr³ = 11,450 voxels
1 voxel = (14μm)³ = 2,744 μm³
実体積 = 11,450 × 2,744 = 31,418,800 μm³
直径 d = 2 × ∛(3V/4π) = 2 × ∛(3 × 31,418,800 / 4π) ≈
